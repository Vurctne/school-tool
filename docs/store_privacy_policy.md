# School Tool — Privacy Policy

**Last updated:** 2026-04-28
**Publisher:** Vurctne

---

## Plain English summary

School Tool is a desktop productivity app. It runs entirely on your computer. We do not
collect, store, transmit, or share any of your data, your files, your school's data, or any
personal information.

---

## What School Tool processes

School Tool reads files you select (CASES21 PDF reports, Master Budget XLSX templates,
SRP PDF exports, prior-period comment workbooks) and writes formatted output workbooks to
folders you choose. Every step runs locally on your Windows device. No data is uploaded to
any server.

---

## What School Tool does NOT do

- We do **not** require an account, sign-in, registration, or any form of identification.
- We do **not** collect, store, or transmit the contents of any file you open.
- We do **not** collect your name, email, IP address, device identifier, or any other
  personal information.
- We do **not** include analytics, telemetry, crash reporters, or third-party tracking SDKs.
- We do **not** require an internet connection to use any feature.
- We do **not** sell, share, or disclose any information to third parties — because we don't
  collect any in the first place.

## Optional: encrypted SIN remembering (HYIA Transfer Code Generator)

The HYIA Transfer Code Generator includes an opt-in feature called "Remember SIN". When
enabled, your Sub-Investment Number is encrypted using the Windows Data Protection API
(DPAPI) and stored in a local file at:

```
%LOCALAPPDATA%\Packages\<MSIX-package-family>\LocalCache\
```

Two important properties:

- The encrypted file can only be decrypted by the same Windows user account on the same
  device that wrote it. It cannot be read by other users, by other devices, or by
  School Tool itself if you sign into Windows as a different user.
- The feature is **off by default**. You enable it explicitly via a checkbox in the HYIA
  tool. Disable it at any time and the file is removed.

The SIN is never transmitted to any server, even when this feature is enabled.
