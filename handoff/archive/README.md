# Handoff archive

Round-by-round handoff docs from earlier eras of this project. Kept
for forensic / historical reference; not part of the active work
surface. Newer (still-active) handoff docs live in the parent
`handoff/` directory.

## Contents

* `phase*.md` — M2 / M3 milestone-era handoffs (Apr 2026) from the
  Orchestrator + Sonnet sub-agent working method.
* `round7_*` through `round50_*` — Apr–early May 2026 work spanning
  the M3 ship and the initial Sub-Program tool features (threshold
  sliders, draggable rail, plain-English errors, MSIX packaging, Store
  submission, prior-period commentary, variance / pacing / watchlist,
  bridge waterfall, etc.). Most were superseded by the Round 51–56
  Sub-Program redesign documented in the parent `handoff/`.

To reference an archived round from a current round's doc, link
through the `archive/` subpath: `[round 45](archive/round45_phaseA_variance_pacing.md)`.
